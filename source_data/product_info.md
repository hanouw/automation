# product_info.md: 탑핀 (Top Pin) 서비스 정보

## 1. 서비스 개요
* **브랜드명**: 탑핀 (Top Pin)
* **서비스 성격**: 공식 PG 결제 시스템 기반의 실시간 상품권 구매 플랫폼
* **핵심 가치**: 신속성(즉시 발매), 안전성(공식 PG), 편의성(신분증 무요구)

---

## 2. 주요 특장점 (USPs)
| 기능 | 상세 설명 |
| :--- | :--- |
| **개인정보 보호** | 신분증이나 카드 사진 등 민감한 개인정보 제출이 전혀 필요 없음 |
| **공식 PG 결제** | 신뢰할 수 있는 공식 결제창을 사용하며, 일반 쇼핑몰과 동일한 카드 결제 프로세스 제공 |
| **실시간 발송** | 결제 완료 즉시 자동화 시스템을 통해 몇 초 내로 PIN 번호 발송 |
| **100% 정품** | 실제 발행된 정품 상품권만 취급하여 환급형 사기나 가짜 거래 위험 원천 차단 |

---

## 3. 취급 상품권 종류
* **문화상품권** (컬쳐랜드 등)
* **구글 기프트카드** (Google Play Gift Card)
* **게임 캐시** (다양한 게임 유료 재화)
* **기타 디지털 상품권** (다양한 활용도의 디지털 핀 번호)

---

## 4. 실제 이용 프로세스 (4-STEP)
1.  **상품 선택**: 원하는 상품권 종류와 금액권을 장바구니에 담습니다.
2.  **결제 진행**: 공식 PG 창을 통해 카드 결제를 완료합니다. (별도 인증 대기 없음)
3.  **PIN 자동 발송**: 시스템이 결제를 확인하는 즉시 핀 번호를 전송합니다.
4.  **즉시 사용**: 수신한 핀 번호를 해당 플랫폼에 등록하여 즉시 사용합니다.

---

## 5. 보안 및 신뢰 요소 (Writing Tips)
* **강조점**: '3분 이내'의 빠른 거래 시간과 '개인 거래의 위험성' 대비 안전함을 대조하여 서술.
* **키워드**: 무서류, 무방문, 자동 발송, 정품 보장, PG 안전 결제.
* **주의사항**: 출처가 불분명한 개인 거래 사이트와의 차별점을 부각하여 이용자 신뢰도 제고.

---

## 6. AI 글쓰기 가이드 (Tone & Manner)
* **톤**: 신뢰감 있고 간결한 어조.
* **타겟**: 빠르고 안전하게 상품권 구매를 원하는 게임 유저 및 온라인 쇼핑객.
* **메시지 전략**: "복잡한 인증 없이, 카드만 있으면 바로 사는 정품 상품권"임을 반복 노출.
