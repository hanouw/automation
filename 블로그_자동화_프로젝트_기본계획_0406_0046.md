# 블로그 자동화 프로젝트 기본 계획 (2026-04-06)

## 프로젝트 목표
- 주제 및 키워드를 입력하면 자동으로 블로그 글 3개 생성 (네이버, 구글, 티스토리 맞춤형)
- 이미지 생성 (키워드 포함 한글 텍스트, 나노바나나 3장)
- 각 블로그 자동 업로드 및 매일 1회 실행

## 현재 진행 상태
- 기술 스택 확인 및 상세 설계를 위한 질문 단계 (답변 대기 중)

## 질문 사항 (사용자 답변 대기)
1. **나노바나나:** 이미지 생성 도구/API의 상세 정보
2. **글 생성 AI:** 선호하는 모델 (Gemini, GPT 등)
3. **블로그 API 설정:** 네이버/티스토리/구글(Blogger) API 권한 및 Client ID 정보
4. **개발 언어:** Python 사용 여부

## 명령 규칙
- "불러와"라고 입력하면 AI는 마지막으로 한 말을 다시 반복함.
